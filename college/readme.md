# College Database - with Node and MongoDB

## API Reference

#### Students

- Register Student

```http
  POST /api/student/register
```

| Parameter | Type     | Description |
| :-------- | :------- | :---------- |
| `email`   | `string` | **Unique**  |

#### Admin

- Get All Users

```http
  GET /api/admin/getUsers
```

- Login User

```http
  POST /api/admin/login
```
